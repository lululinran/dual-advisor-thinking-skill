# 双顾问思维（Dual Advisor Thinking）

一个用于AI Agent 的技能（Skill），通过两位精心选择的顾问，让开放性思考不再平淡，更有实际价值。

## 这是什么

当你面对需要判断、策略、创意方向或深度思考的问题时，本技能会：

1. 选择两位在该话题上有真实分歧的顾问
2. 从各自视角分析问题
3. 暴露两者之间的真实冲突
4. 做出清晰的综合判断（而非模糊的折中）
5. 给出可执行的下一步

目标不是角色扮演，而是将答案锚定在两条相关的知识传统中。

## 适用场景

- 战略决策和职业选择
- 产品方向和创意判断
- 个人认知系统搭建
- 写作选题和传播策略
- 任何"你怎么看"类的开放性问题

**不适用**：简单事实查询、机械格式化、纯翻译任务、代码执行。

## 安装

```bash
npx skills add <owner>/dual-advisor-thinking -g -y
```

## 使用方式

在支持技能的 AI Agent 环境中，直接提问即可触发：

```
我该不该辞职做独立产品？
这个公众号选题有没有意思？
我想搭一个长期生长的个人认知系统。
```

也可以提到触发词：**名人顾问**、**双顾问**、**历史名人顾问**、**橙子去皮**。

## 输出结构

```
两位顾问
顾问A的视角
顾问B的视角
有效冲突
我的综合判断
下一步
```

## 顾问选择原则

- 优先相关性，而非知名度
- 避免同侧配对（两位观点相近的顾问）
- 内置中文语境支持（曾国藩、王阳明、鲁迅、韩非子等）
- 配对模式：系统构建者 vs 实践者、怀疑论者 vs 远见者、艺术家 vs 工程师……
