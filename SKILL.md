# 双顾问思维

## 概述

借助两位精心选择的顾问，让开放性思考不再平淡，更有实际价值。目标不是角色扮演，而是将答案锚定在两条相关的知识传统中，暴露其分歧，并做出具体判断。

## 工作流程

1. **路由**：仅在用户寻求判断、解读、策略、品味、方向或反思性思考时使用本技能。若只是简单提问，直接回答即可。
2. **框架**：识别决策类型、背景、隐含的权衡、时间跨度，以及对用户而言"更好"意味着什么。若请求过于模糊，提一个澄清性问题，或明确陈述假设。
3. **选择恰好两位顾问**，除非用户另有要求。选择真实人物，其作品、方法或决策记录与主题高度相关。
4. **制造张力**。两位顾问应产生有建设性的分歧：系统构建者 vs 实践者、怀疑论者 vs 远见者、艺术家 vs 工程师、道德主义者 vs 战略家、历史学家 vs 未来学家。
5. **用用户的语言回答**。简要说明配对理由，然后呈现每个视角、两者的冲突、你的综合判断，以及下一步行动。
6. **做出判断**。不要平均两种观点；说明在用户的约束条件下哪个视角应占主导，并说明原因。

## 顾问规则

- 优先考虑相关性，而非知名度。除非契合度特别强，否则避免默认选择爱因斯坦、乔布斯、巴菲特或孙子。
- 避免同侧配对。若两位顾问会给出相似建议，用一位有用的对立者替换其中一位。
- 谨慎使用在世公众人物。依赖其广为人知的作品和公开方法，而非不确定的私人信仰或当下言论。
- 不捏造直接引语、私人信仰或具体历史断言。
- 除非用户明确要求角色扮演，否则不模仿顾问本人。
- 使用"从X的视角来看"、"类似Y的批评"或"这一传统会强调"等表述方式。
- 若话题依赖当前事实、价格、法律、医疗建议或其他高风险/时效性信息，在进行顾问分析之前先用工具核实事实。

## 输出结构

对于实质性回答，使用以下结构：

```markdown
**两位顾问**
[顾问A] + [顾问B]，简要说明配对理由及张力所在。

**顾问A的视角**
[该视角关注什么、重视什么、警惕什么。]

**顾问B的视角**
[该视角关注什么、重视什么、警惕什么。]

**有效冲突**
[将两者放在一起所暴露的真实分歧、权衡或盲点。]

**我的综合判断**
[清晰的判断，而非中立的折中。说明在当前情境下哪个视角应占主导。]

**下一步**
[具体行动、测试、决策规则或更好的问题。]
```

对于小问题，在保留顾问选择、冲突、综合判断和下一步的前提下，将结构压缩为几段话。

## 质量标准

完成前检查：

- 配对足够出人意料，能产生张力，但又不过于生僻而流于装饰。
- 每个视角都改变了答案本身，而不仅仅是措辞。
- 冲突是明确的。
- 综合判断有主心骨：偏向某条路径，说明条件，避免"视情况而定"的模糊。
- 下一步在用户可能的情境下是可执行的。
