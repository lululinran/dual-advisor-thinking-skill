# 回答质量评估标准

在完成重要回答前使用，或当初稿感觉像是穿着戏服的普通提示词回复时使用。

## 五项检查

1. **触发适配**：用户寻求的是思考、判断、方向或讨论。若他们询问的是直接事实或机械任务，不要强行套用本技能。
2. **配对适配**：两位顾问各有不同的相关理由。他们的名字不是装饰。
3. **张力**：两个视角制造了真实的分歧、权衡或盲点。
4. **综合判断**：答案做出了抉择。不是礼貌性的折中。
5. **实用性**：下一步足够具体，用户可以尝试、测试、拒绝或改进。

## 质量等级

| 等级 | 信号 |
|---|---|
| 弱 | 顾问可以被任何名人替换，答案几乎不变。 |
| 合格 | 每位顾问提供了不同视角，但综合判断仍然保守。 |
| 强 | 配对暴露了真实冲突，综合判断做出了可辩护的抉择。 |
| 优秀 | 答案改变了用户对问题的框架，并给出了实际的下一步。 |

## 常见失败模式

- **顾问角色扮演**：写虚假对话或虚假引语。修复方法：描述视角，而非假装成本人。
- **维基百科拖拽**：总结人物传记。修复方法：只使用该人物作品中改变分析的那部分。
- **虚假平衡**：以"两者都重要"结尾。修复方法：说明现在哪个视角应占主导，以及在什么条件下会切换。
- **同侧配对**：两位顾问强化相同的世界观。修复方法：将其中一位换成有建设性的对立者。
- **提示词膨胀**：每次都向用户解释整个方法。修复方法：仅在答案需要时才让方法可见。
- **过度名人默认**：对不相关话题使用相同的名人。修复方法：从问题的权衡出发选择，而非从名气出发。
- **没有下一步**：分析很好但没有牵引力。修复方法：以实验、标准、问题或决策规则结尾。

## 综合判断测试

默默问自己：

- 顾问A会认为顾问B遗漏了什么？
- 顾问B会认为顾问A危险地高估了什么？
- 用户的情境现在使什么变得更重要？
- 什么会改变我的建议？
- 能够产生新信息的最小行动是什么？

## 风格指南

- 匹配用户的语言和期望的信息密度。
- 有观点，但不要变得戏剧化。
- 顾问说明要简短；把更多篇幅用在用户的问题上。
- 当话题敏感时，把锋芒放在推理中，而非语气上。
