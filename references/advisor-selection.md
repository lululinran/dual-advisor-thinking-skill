# 顾问选择

当默认配对不明显，或用户提出高价值的战略、创意或个人问题时，使用此参考。

## 选择算法

1. 提取真实任务：决策、诊断、战略、品味、身份认同、系统设计或意义建构。
2. 识别隐含的权衡：速度 vs 深度、自由 vs 纪律、原创性 vs 可读性、增长 vs 风险、真相 vs 实用性、工艺 vs 传播、伦理 vs 权力。
3. 选顾问A代表用户当前的直觉或该领域的主流逻辑。
4. 选顾问B从不同传统对这种直觉进行压力测试。
5. 若两位顾问基本会达成一致，放弃这个配对。
6. 优先选择有方法论的顾问，而不只是有名气的人。方法论可以是理论、作品体系、运营哲学、艺术实践或决策记录。

## 配对模式

- 系统构建者 + 实践者：适合必须经受现实考验的战略。
- 艺术家 + 编辑/评论家：适合创意方向和公众传播。
- 哲学家 + 实用主义者：适合约束条件下的人生选择和价值判断。
- 科学家/工程师 + 人类行为思考者：适合产品、技术和设计。
- 历史学家 + 未来学家：适合趋势解读和长视野判断。
- 优化者 + 怀疑论者：适合风险、投资、职业赌注和规划。
- 内部人 + 外部人：适合用户可能被局部惯例所困的情况。

## 中文语境

当中国思想家能提供更好的本土框架时使用，而非装饰性地添加。

| 话题 | 有效配对 | 有效张力 |
|---|---|---|
| 务实人生策略 | 曾国藩 + 王阳明 | 通过制度磨砺自律 vs 从内心知行合一 |
| 治理与激励 | 韩非子 + 孔子 | 规则与执行 vs 礼仪与道德培养 |
| 有社会力度的写作 | 鲁迅 + 胡适 | 诊断与激发 vs 清晰与改革 |
| 创业判断 | 任正非 + 彼得·德鲁克 | 生存导向的组织 vs 以贡献为本的管理 |
| 长期耐心 | 司马迁 + 纳西姆·尼古拉斯·塔勒布 | 历史韧性 vs 非线性风险 |

## 劣质配对的信号

- 两位顾问都是与话题没有明确关联的知名通才。
- 配对听起来令人印象深刻，但没有产生冲突。
- 其中一位顾问只是为了名气而存在。
- 答案变成了人物传记，而非分析。
- 所选顾问把话题强行纳入其世界观，而非照亮用户的实际问题。

## 修复动作

- 太平淡：用另一领域的怀疑论者替换一位顾问。
- 太抽象：至少选一位有具体决策记录的建设者/实践者。
- 太商业化：加入人文学者、心理学家、艺术家或伦理学家。
- 太说教：加入实践者、经济学家、战略家或系统思考者。
- 太西方正典默认：考虑中国或非西方框架是否更契合用户和话题。
